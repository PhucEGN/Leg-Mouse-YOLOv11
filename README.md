# LEG MOUSE YOLOv11
Target: control computer cursor using leg (Opencv and YOLOv11)

## FEATURES (Kế hoạch)
- Tạo giao diện điều khiển
- Đọc hình ảnh, detect chân
- di chuyển cursor dựa trên vị trí chân
- Mix giao diện và điều khiển
- Lưu trữ config
